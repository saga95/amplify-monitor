Version history and release notes for Amplify Monitor.

v0.1.22 (February 2025)

Latest

üîÄ Build Comparison

Compare two builds side-by-side to identify what changed:

  • Side-by-Side View - Compare any two builds from the same branch
  • Status Comparison - See how build status changed between runs
  • Duration Analysis - Compare build times to spot performance regressions
  • Error Diff - Visual comparison of errors found in each build
  • Issue Breakdown - See which issues were resolved or newly introduced
  • Quick Actions - View full logs or open in AWS Console from comparison view

v0.1.21 (February 2025)

üéØ Custom Failure Patterns

Define your own error detection patterns for build logs:

  • Pattern Editor - Create patterns with regex or plain text matching
  • Real-Time Testing - Test patterns against sample log output before saving
  • Presets Library - 10+ pre-built patterns for common errors (ESLint, TypeScript, Webpack, NPM, etc.)
  • Import/Export - Share patterns with your team as JSON files
  • Category Labels - Categorize patterns as error, warning, or info
  • Toggle On/Off - Enable/disable patterns without deleting
  • Rich Metadata - Add root cause descriptions and suggested fixes
  • Duplicate Patterns - Copy existing patterns as starting point

New Commands

  • Amplify Monitor: Manage Custom Failure Patterns - Opens the Custom Patterns panel

v0.1.20 (February 2025)

üë• Multi-Account AWS Profile Manager

Seamlessly work across multiple AWS accounts without leaving VS Code:

  • Profile Discovery - Auto-discovers all profiles from ~/.aws/credentials and ~/.aws/config
  • Quick Switch - Change active profile with one click from status bar or Profile Manager panel
  • Credential Validation - Test credentials for each profile, see account ID and user/role
  • Visual Status - Active profile clearly marked with orange badge, validation badges for all profiles
  • Add New Profiles - Create new AWS profiles directly in VS Code with guided form
  • Edit Config Files - Quick access to credentials and config files for manual editing
  • SSO Support - Full support for AWS SSO profiles and assumed roles
  • Region Per Profile - Each profile shows its configured default region
  • AWS Console Integration - One-click launch to AWS Console with correct region
  • Status Bar Indicator - Current profile always visible, click to quick-switch

New Commands

  • Amplify Monitor: Manage AWS Profiles - Opens the full Profile Manager panel
  • Amplify Monitor: Switch AWS Profile - Quick pick dropdown to switch profiles

v0.1.19 (February 2025)

üåê Custom Domain Validator

New feature to validate custom domain configurations:

  • DNS Record Validation - Checks CNAME, A, TXT, CAA records for proper Amplify setup
  • SSL Certificate Check - Verifies certificate validity, expiration, and issuer
  • Apex Domain Detection - Identifies apex vs subdomain and provides correct guidance
  • CloudFront Detection - Validates CDN configuration and CNAME targets

v0.1.18 (February 2026)

üîß Env Vars Troubleshooter

Comprehensive environment variable analysis:

  • Code Scanning - Finds all process.env.* and import.meta.env.* references
  • Missing Detection - Identifies vars used in code but not defined
  • Amplify Sync Check - Warns if local .env vars aren't in Amplify
  • Framework-Aware - Understands Next.js, Vite, CRA prefix requirements
  • Security Scanning - Detects hardcoded secrets and exposed sensitive vars
  • Client-Side Exposure - Warns about sensitive vars with NEXT_PUBLIC_ prefix
  • Gitignore Validation - Ensures .env files aren't committed
