Quick Start

Get up and running in under 2 minutes

1

Install the Extension

Search for "Amplify Monitor" in VS Code Extensions, or click the install button above.

ext install SagaraHarasgama.amplify-monitor
2

Configure AWS Credentials

Set up your AWS credentials using environment variables or AWS CLI profile.

aws configure
3

Open Your Project

Open any Amplify project — the extension auto-detects and connects to your apps.

4

Start Monitoring!

Click the Amplify Monitor icon in the Activity Bar and open the Dashboard.
