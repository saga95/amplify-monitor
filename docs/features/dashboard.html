Overview

The Dashboard provides a visual overview of all your AWS Amplify apps, branches, and build statuses. No more switching between AWS Console tabs ‚Äî see everything at a glance.

Opening the Dashboard

  1. Press Ctrl+Shift+P (or Cmd+Shift+P on Mac)
  2. Type "Amplify Monitor: Open Dashboard"
  3. Press Enter

Or click the Dashboard icon (üìä) in the Amplify Monitor Activity Bar.

Features

Summary Bar

The top of the dashboard shows aggregate statistics:

  • Total Apps - Number of Amplify apps across all regions
  • Succeeded - Apps with successful latest builds (green)
  • Failed - Apps with failed latest builds (red)
  • Running - Currently building apps (blue)

App Cards

Each app is displayed as a card showing:

  • App name and ID
  • Branch name
  • Build status (with color coding)
  • Last build time (relative, e.g., "5m ago")
  • Quick action buttons

Filtering & Search

Use the filter bar to:

  • Search by app name
  • Filter by status (All, Succeeded, Failed, Running)
  • Sort by name, status, or last build time

Auto-Refresh

Enable auto-refresh in settings to keep the dashboard updated:

{
  "amplifyMonitor.autoRefresh": true,
  "amplifyMonitor.autoRefreshInterval": 60
}

Tips

Pro Tips
  • Click any card to expand details and see recent build history
  • Use keyboard shortcuts: R to refresh, F to filter
  • Right-click a card for context menu options
