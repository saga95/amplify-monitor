Commands Reference

Complete list of all Amplify Monitor commands. Access via Command Palette (Ctrl+Shift+P).

Dashboard & Overview

Command Description
Amplify Monitor: Open Dashboard Opens the visual portfolio dashboard showing all apps, branches, and build statuses
Amplify Monitor: List Apps Refreshes the apps list in the sidebar tree view
Amplify Monitor: Switch AWS Profile Change the AWS profile used for authentication

Build Diagnosis

Command Description
Amplify Monitor: Diagnose Latest Failed Build Analyzes the most recent failed job and identifies root causes with suggested fixes
Amplify Monitor: Apply Quick Fix Shows all available quick fixes and applies the selected one

Troubleshooting Tools

Command Description
Amplify Monitor: Troubleshoot Environment Variables Scans code for env var usage, detects missing/exposed vars, finds hardcoded secrets
Amplify Monitor: Detect Node Version Issues Finds Node.js version mismatches between local, package.json, and Amplify
Amplify Monitor: Validate Custom Domain Validates DNS records, SSL certificates, and Amplify configuration for custom domains
Amplify Monitor: Pre-Deploy Validation Runs pre-deployment checks: git status, dependencies, build validation, env checks

Build Optimization

Command Description
Amplify Monitor: Build Optimization Wizard Guided wizard to optimize build speed with caching, dependency, and config improvements
Amplify Monitor: Analyze Bundle Size Analyzes build output size vs Amplify's 230MB limit with recommendations
Amplify Monitor: Show Build Performance Displays historical build time trends and performance metrics
Amplify Monitor: Detect Monorepo Structure Auto-detects monorepo setup and generates appropriate amplify.yml configuration

Notifications

Command Description
Amplify Monitor: Configure Notifications Set up webhook notifications for Slack, Microsoft Teams, Discord, or custom endpoints

Secrets Management

Command Description
Amplify Monitor: Manage Secrets Browse AWS SSM Parameters and Secrets Manager, sync to Amplify env vars

Build Management

Command Description
Amplify Monitor: Start Build Triggers a new deployment for the selected app and branch
Amplify Monitor: Stop Build Cancels a running deployment
Amplify Monitor: Open in AWS Console Opens the app in the AWS Amplify Console in your browser

Environment Variables

Command Description
Amplify Monitor: Add Environment Variable Adds a new environment variable to a branch
Amplify Monitor: Edit Environment Variable Modifies an existing environment variable
Amplify Monitor: Delete Environment Variable Removes an environment variable from a branch
Amplify Monitor: Reveal Environment Variable Shows the masked value of an environment variable

Gen2 Migration

Command Description
Amplify Monitor: Analyze Gen1 → Gen2 Migration Analyzes your Gen1 project for Gen2 compatibility and provides migration guidance

Keyboard Shortcuts

You can assign custom keyboard shortcuts to any command:

  1. Open Keyboard Shortcuts (Ctrl+K Ctrl+S)
  2. Search for "Amplify Monitor"
  3. Click the + icon to add a keybinding

Suggested Shortcuts

Command Suggested Shortcut
Open Dashboard Ctrl+Alt+A D
Diagnose Latest Failed Build Ctrl+Alt+A F
Start Build Ctrl+Alt+A S
Quick Access

All commands are also available from the context menu when right-clicking in the Amplify Monitor tree views.
