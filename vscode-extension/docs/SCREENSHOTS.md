# Screenshots for Marketplace

Add these screenshots to improve your marketplace listing:

## Required Screenshots

1. **screenshot.png** (1280x800 recommended)
   - Show the full extension with Apps, Jobs, and Diagnosis panels
   - Use a dark theme for consistency with AWS branding

2. **screenshot-diagnosis.png** (800x600)
   - Focus on the Diagnosis panel
   - Show detected issues with expanded fixes

3. **screenshot-apps.png** (600x400)
   - Show the Apps tree with multiple apps/branches

## How to Capture

1. Open VS Code with the extension
2. Configure with real Amplify apps
3. Run diagnosis on a failed build
4. Use VS Code's screenshot tools or Windows Snipping Tool
5. Crop to focus on relevant areas

## Tips

- Use consistent window sizes
- Hide personal information (app IDs can stay)
- Use high contrast for readability
- Add subtle annotations if needed
